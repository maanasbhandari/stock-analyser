# 📈 StockSense — Stock Investment Analyzer

A sleek, finance-themed web app that analyzes any stock ticker against 10 key investment parameters and gives a clear **Invest / Caution / Skip** verdict.

## Features
- 🔍 Search any stock by ticker (AAPL, INFY, RELIANCE.NS, etc.)
- 📊 Evaluates 10 parameters: P/E, Debt/Equity, ROE, Profit Margin, Revenue Growth, Current Ratio, P/B, EPS, Dividend Yield, Beta
- ✅ Clear verdict with score bar
- 📱 Fully responsive dark-themed UI
- ⚡ Live data via Yahoo Finance

## How to run
Just open `index.html` in your browser — no server needed!

## Data source
Yahoo Finance public API via allorigins CORS proxy.
